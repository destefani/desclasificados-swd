## Research Methodology

All research should follow these guidelines:

1. **Source Documents**: Always reference specific document IDs from `data/generated_transcripts/`
2. **Citations**: Include document_id, date, and relevant metadata
3. **Cross-references**: Link related findings across themes and cases
4. **Verification**: Note confidence levels and corroborating sources
5. **Gaps**: Document what information is missing or redacted
