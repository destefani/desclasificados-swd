# Chilean Timeline (1964-1990)

A chronological overview of key events for interpreting declassified documents.

---

## Frei Government (1964-1970)

### 1964

| Date | Event |
|------|-------|
| September 4 | **Eduardo Frei elected President** (PDC) with CIA support |
| | CIA spent ~$2.6 million to support Frei against Allende |

### 1965

| Date | Event |
|------|-------|
| August 15 | MIR (Movement of Revolutionary Left) founded |

### 1967

| Date | Event |
|------|-------|
| | Land reform law passed |
| | "Chilenization" of copper (partial nationalization) |

---

## Allende Election Crisis (1970)

### Before Election

| Date | Event |
|------|-------|
| March | 40 Committee approves anti-Allende "spoiling" operation |
| June 27 | Kissinger: "I don't see why we need to stand by and watch a country go communist" |

### September 1970

| Date | Event |
|------|-------|
| September 4 | **Allende wins election** (36.3%) - plurality, not majority |
| September 8 | 40 Committee meets on Chile response |
| September 14 | 40 Committee authorizes Track I (political action) |
| September 15 | **Nixon orders Track II** - "make the economy scream" |
| | $10 million authorized; Track II kept from State Dept and Ambassador |

### October 1970

| Date | Event |
|------|-------|
| October 15 | CIA passes weapons to Chilean coup plotters |
| October 19 | First kidnapping attempt on General Schneider fails |
| October 22 | **General René Schneider assassinated** in botched kidnapping |
| October 24 | Congress confirms Allende as President |

### November 1970

| Date | Event |
|------|-------|
| November 3 | **Allende inaugurated** |

---

## Allende Government (1970-1973)

### 1971

| Date | Event |
|------|-------|
| July | Congress unanimously approves copper nationalization |
| September 9 | 40 Committee approves $700,000 for El Mercurio |
| | Australian intelligence (ASIS) establishes Santiago station to support CIA |

### 1972

| Date | Event |
|------|-------|
| April 11 | 40 Committee adds $965,000 for El Mercurio |
| October | Truckers' strike (supported by CIA funding) |
| | Economic crisis deepens |

### 1973

| Date | Event |
|------|-------|
| March | Congressional elections - UP gains but no majority |
| June 29 | **Tanquetazo** - failed coup attempt by tank regiment |
| August 22 | Chamber of Deputies declares Allende government unconstitutional |
| August 23 | **General Carlos Prats resigns** as Army commander |
| August 23 | **Pinochet appointed** Army Commander by Allende |
| September 8 | Pinochet informed of coup plans |
| September 9 | Final coup planning |

---

## The Coup: September 11, 1973

| Time | Event |
|------|-------|
| 6:00 AM | Navy seizes Valparaíso |
| 7:00 AM | Military issues ultimatum to Allende |
| 8:00 AM | Allende arrives at La Moneda palace |
| 9:10 AM | Allende's final radio address |
| 11:52 AM | **Air Force bombs La Moneda** |
| ~2:00 PM | **Allende dies** (officially suicide) |
| Evening | Military junta announces takeover |

---

## Military Dictatorship (1973-1990)

### 1973 (Post-Coup)

| Date | Event |
|------|-------|
| September 12 | US recognizes junta |
| September 13 | Congress dissolved |
| September-October | Mass arrests, stadium detentions |
| October | **Caravan of Death** - Arellano Stark executes 75+ prisoners |
| November | DINA established as Army intelligence unit |

### 1974

| Date | Event |
|------|-------|
| February | Pinochet requests CIA assistance for DINA |
| March | Contreras visits CIA headquarters |
| June 14 | **DINA officially established** (Decree 521) |
| September 30 | **Carlos Prats assassinated** in Buenos Aires (DINA) |
| October 5 | **Miguel Enríquez (MIR leader) killed** by DINA |
| December | Pinochet declared "Supreme Chief of the Nation" |

### 1975

| Date | Event |
|------|-------|
| Mid-year | CIA briefly puts Contreras on payroll |
| October 6 | **Bernardo Leighton shot** in Rome (survives) |
| November 25 | **Operation Condor formally established** in Santiago |
| December 4 | Church Committee begins public hearings on Chile |

### 1976

| Date | Event |
|------|-------|
| September 21 | **Orlando Letelier assassinated** in Washington DC |
| | Ronni Moffitt also killed |
| | FBI investigation begins |

### 1977

| Date | Event |
|------|-------|
| August | **DINA dissolved**, replaced by CNI |
| | Contreras removed, Mena takes over |
| | Letelier case creates diplomatic crisis |

### 1978

| Date | Event |
|------|-------|
| April | Amnesty law passed (self-amnesty for 1973-1978 crimes) |
| July | **Gustavo Leigh dismissed** from junta for opposing Pinochet |

### 1980

| Date | Event |
|------|-------|
| September 11 | **New constitution approved** (plebiscite under dictatorship) |
| | Pinochet's rule institutionalized |
| | Plebiscite scheduled for 1988 |

### 1982-1983

| Date | Event |
|------|-------|
| 1982 | Economic crisis hits Chile |
| 1983 | Mass protests begin |
| | Political parties begin to reorganize |

### 1986

| Date | Event |
|------|-------|
| September 7 | **Assassination attempt on Pinochet** fails |
| | Manuel Rodríguez Patriotic Front attack |
| | 5 bodyguards killed, Pinochet unhurt |

### 1988

| Date | Event |
|------|-------|
| February 2 | Opposition coalition (Concertación) formed |
| October 5 | **Plebiscite: NO wins** (55% to 43%) |
| | Pinochet forced to accept transition |

### 1989

| Date | Event |
|------|-------|
| December 14 | **Patricio Aylwin elected President** |
| | First democratic election since 1970 |

### 1990

| Date | Event |
|------|-------|
| February 22 | CNI dissolved |
| March 11 | **Pinochet leaves presidency** |
| | Aylwin inaugurated |
| | Pinochet remains Army Commander until 1998 |

| Date | Event |

## Document Corpus Date Distribution

From Phase 1 analysis, document dates cluster around:

| Year | Count | Significance |
|------|-------|--------------|
| **1970** | 68 | Allende election (peak) |
| 1971 | 21 | Early Allende government |
| 1972 | 29 | Destabilization period |
| **1973** | 23 | Coup year |
| 1974-1978 | Sparse | DINA period (underrepresented) |
| **1991** | 72 | Declassification processing date |

**Note**: Many 1991 dates may reflect when documents were processed for declassification, not when they were created.

## Key Periods for Research

| Period | Focus | Key Events |
|--------|-------|------------|
| **1970** | Election intervention | Track I, Track II, Schneider |
| **1970-1973** | Destabilization | CIA funding, economic pressure |
| **Sept 1973** | Coup | Military takeover, Allende death |
| **1973-1977** | DINA period | Maximum repression, Condor |
| **1976** | Letelier case | Washington assassination |
| **1988-1990** | Transition | Plebiscite, democratization |
